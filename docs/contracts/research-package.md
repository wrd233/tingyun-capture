# Research Package Contract

`README_FOR_RESEARCH.md` is the single Task entry point. It states goal, success criteria, prohibited assumptions, Sessions, reading order, Raw/Derived and observation/inference boundaries, gaps, and security level without answering the research question.

Task content includes metadata, summary/timeline, Session index, navigation/endpoint/correlation indexes, evidence gaps, success-criteria evidence presence, Session AI-ready evidence, download indexes, integrity, validation, and promotion input. Criterion statuses are only EVIDENCE_PRESENT, EVIDENCE_MISSING, or UNKNOWN.
