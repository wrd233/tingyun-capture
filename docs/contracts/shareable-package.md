# Shareable Package Contract

Shareable is generated from an empty directory using an allowlist. Allowed classes are the research entry point, redacted Task/research indexes, strictly redacted Session analysis, tokenized navigation/correlation/endpoint observations, redacted normalized CSV, and integrity/omission/security/validation reports. Raw, profiles, environments, shell history, private mappings, absolute local paths, and credentials are forbidden.

Directory and ZIP scans return PASS or BLOCKED. Any finding blocks and removes the ZIP. Repeated frozen input must have the same file set, core file content, ordering, token mapping, navigation index, and correlation candidates.
