# Correlation Candidate Contract

`kind` is `exact_value_match`; `relation_status` is `CANDIDATE_ONLY`. Source and target identify event and field path. Shareable output contains value token/type/hash, not the scalar. Scope is same Interaction Window or the adjacent bounded page window. Candidate ID, token assignment, sorting, and output are deterministic. The contract forbids lineage, dependency, cause, and multi-hop inference.
