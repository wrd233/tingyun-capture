## 1. 决策冲突消解

### 1.1 Session 开始字段

早期曾建议强制“一句话目标”，最终被 Q035 覆盖：

> 只要求 Session 名称。

### 1.2 AI-ready 生成

正常 SEALED：自动生成。

INTERRUPTED：不自动生成，必须用户后续明确封存。

### 1.3 目标 Origin 外行为

早期考虑轻量跟踪，最终 Q057 选择最简单方案：

> 完全忽略 Origin 外内容。

### 1.4 Timing

Q064 选择 B，但明确做减法：

- 基础时间是核心。
- 更细 Timing 仅低成本附带。
- 不进入产品重点。

## 2. 设计和实现阶段的使用方式

每个主要设计决策应能回指本台账。

如果实现 Agent 计划加入新能力，应先检查：

- 是否已在 Q090 非目标中明确禁止。
- 是否与 Q091 Greenfield 原则冲突。
- 是否改变 Q001 的产品边界。
- 是否破坏 Q045 的“Capture 不主动操作页面”。
- 是否破坏 Q027 的“运行中零 LLM”。
